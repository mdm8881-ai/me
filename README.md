# 자기소개

## About Me
![](wordcloud_1776066158909.png)

&nbsp;&nbsp;&nbsp;&nbsp;
저는 언어를 사랑하는 사람으로서 통역과 번역 분야에 깊은 관심을 가지고 있습니다. 서로 다른 언어를 사용하는 각국의 사람들이 의미를 정확하게 전달하도록 돕는 과정은 단순한 언어 변환을 넘어 사람과 사람, 문화와 문화를 연결하는 중요한 역할을 한다고 생각합니다. 특히 저의 모국어가 아닌 언어로 사람들과 소통할 때 느끼는 성취감은 저에게 큰 동기가 되었습니다. 

&nbsp;&nbsp;&nbsp;&nbsp;
이와 관련하여 저는 “AI 통역과 인간 통역은 어떤 차이를 가지는가?”라는 질문을 떠올려보았습니다. AI는 인간과 비교해 보았을 때 매우 빠른 속도와 정확성을 제공할 수 있지만, 감정의 미묘한 표현이나 문화적, 시대적 맥락을 반영하는 데에는 한계가 있다고 생각합니다. 이를 확인하기 위해 국제 회의 기록, 뉴스 기사, 인터뷰 자료 등을 활용하여 AI 번역과 인간 통역 결과를 비교하고, 의미 전달과 문맥 자연스러움의 차이를 분석할 계획입니다.

&nbsp;&nbsp;&nbsp;&nbsp;
또한 단순히 결과를 비교하는 데서 그치지 않고, 통번역 방식의 유형을 분류하여 어떤 상황에서 AI가 더 강점을 보이고 어떤 상황에서 인간 통역이 더 적합한지 정리해 보고자 합니다. 이를 통해 두 통역 방식의 장단점을 상호 보완적으로 이해할 수 있을 것이라 생각합니다.

&nbsp;&nbsp;&nbsp;&nbsp;
더불어 조사 결과를 바탕으로, AI를 단순한 번역 도구가 아니라 보조 수단으로서 활용하여 결과물을 점검하고 수정하는 방식으로 효율성과 정확성을 함께 높이고자 합니다.

&nbsp;&nbsp;&nbsp;&nbsp;
저의 궁극적인 목표는 5년 뒤 UN과 같은 국제기구에서 통역사로 활동하는 것입니다. AI 기술을 활용하되 인간만이 표현할 수 있는 감정과 뉘앙스를 살려, 언어를 통해 사람과 문화를 연결하는 역할을 하고 싶습니다.
